# Primeira aplicação com Angular

### Instalação do Angular
npm install -g @angular/cli

### Comandos utilizados em aula
- <strong>ng new <i>nome-da-aplicacao</i></strong>
<br>Cria um projeto Angular

- <strong>ng serve</strong>
<br>Roda a aplicação
